# CatStealer API

## How to run

1. **Clone the repository**
2. **Create a `.env` file** in the root directory:
   ```env
   SA_PASSWORD=YourStrongPassword123!
   CAAS_API_KEY=your-cat-api-key-here
   ```
3. **Run the container:**
   ```bash
   docker-compose up -d
   ```
4. **Access the API at:** http://localhost:5000
